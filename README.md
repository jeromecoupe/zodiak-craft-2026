# Craft Zodiak Project 2026

A fake agency website to learn Craft CMS

## Installation

- clone repository
- install [DDEV](https://ddev.com/)
- install Docker provider
- run `composer install`
- run `npm install`
- bring back uploaded assets from production and put them in `./web/uploads`
- bring back a database dump and import it in ddev
- create and configure `.env` follwing the `.env.example` file
- run `npm run build`
- run `ddev start`
- you should have a developement environement running
